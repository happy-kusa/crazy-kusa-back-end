### 執行

- 編譯 + 執行: `cargo run`
- 編譯: `cargo build`
- 編譯(發布用): `cargo build --release`
- 檢查是否能編譯: `cargo check`

### 命名規則

- 全域變數 環境變數 靜態 常數 => 全大寫用\_分開
- 變數 => 全小寫用\_分開
- 類別 特徵 結構 列舉 => 大寫駝峰
- 函數 方法 模組 檔案 => 全小寫用\_分開
- 目錄 => 全小寫用\-分開
